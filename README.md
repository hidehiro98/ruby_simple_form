# Ruby simple CGI server
## Technology
- WEBrick
- CGI

## How to use
- `git clone git@github.com:hidehiro98/ruby_simple_form.git`
- `ruby_simple_form`
- `ruby webrick.rb`
- access these addresses
  - http://localhost:8080/
  - http://localhost:8080/params?a=hoge
  - http://localhost:8080/cgi?a=hoge
